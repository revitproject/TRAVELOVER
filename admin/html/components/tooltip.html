Tooltip

네트워크 이스케이프 해치의 사용 설정 여부입니다. 부팅 시 네트워크를 연결할 수 없는 경우, 이스케이프 해치는 기기 정책을 새로고침하기 위해 일시적으로 네트워크에 연결하라는 메시지를 사용자에게 표시합니다. 정책을 적용하면 적절한 네트워크가 없고 기기가 작업 잠금 모드에서 앱으로 부팅되거나 사용자가 기기 설정에 도달할 수 없는 경우 네트워크에 연결할 수 없는 문제를 방지할 수 있습니다.

      <div class="tooltip-content">
        <strong class="tooltip-title">툴팁 타이틀</strong>

      </div>
      <button type="button" class="tooltip-close"><span class="hidden">닫기</span></button>
    </div>
  </div>
