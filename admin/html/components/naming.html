Naming

CSS Class Naming

  • Kebab casing 으로 영문 소문자, 숫자, hyphen(-)으로 조합하여 작성한다.
  • hyphen(-)조합은 최대 3단계를 초과하지 않는 걸 권장한다.
  • 단어와 숫자 조합은 hyphen(-)을 생략한다.
  • 숫자는 1부터 시작하며 '01' 과 같이 앞에 '0'을 붙여 사용하지 않는다.
  • Multi class의 조합은 3개 이하(우선순위 점수 33점 미만)을 권장한다.
  • 스타일 지정 시 ID 선택자의 사용을 지양한다.
  • 엘리먼트에 직접적인 스타일을 적용하지 않고, 모든 스타일은 클래스를 통해 지정한다.
name-name

.list2
.btn-mix-outlined-s
.btn-mix-outlined .on

Reserve a Prefix

예약된 접두사로 기본적으로 공통 사용하는 CSS naming

Reserve a Prefix Description
.base- 페이지내에서 유니크하게 하나만 사용하는 기본적인 구조, 레이아웃 요소나 페이지 필수요소에 사용
base-wrap, base-header, base-content, base-aside
.btn- button 관련된 스타일
.bul- bullet에 관련된 스타일
.area- .wrap 요소를 그룹으로 묶어주는 스타일
area-dropdown
.wrap- 독립적인 item요소를 그룹으로 묶어주는 스타일
wrap-group, wrap-searchbox
.form- 입력양식의 레이아웃 스타일
.inp- input, select, textarea에 관련된 스타일
.lb- radio,checkbox의 label 스타일
.ui- 자바스크립트 관련 클래스
